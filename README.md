# CityMind — An Urban Intelligence System

An AI-powered smart city simulation system built for the AI2002 course at FAST-NUCES.
CityMind helps city authorities make smarter, faster, and better-informed decisions
using a combination of AI search, optimization, and machine learning techniques.

## 🏙️ Project Overview
The city is modeled as a grid-based graph where locations are nodes and roads are edges.
The system addresses five core urban challenges in an integrated simulation.

## ⚙️ Features
1. **City Layout Planning** — Constraint-based placement of city zones
2. **Road Network Optimization** — Minimum cost road network with redundancy
3. **Ambulance Placement** — Optimized emergency vehicle positioning
4. **Emergency Routing** — Dynamic real-time rerouting under changing conditions
5. **Crime Risk Prediction** — ML-based risk classification integrated into the city graph

## 🛠️ Tech Stack
- Python
- Flask
- scikit-learn
- Machine Learning (clustering + classification)

## 🚀 How to Run
```bash
1. Install dependencies:
pip install flask scikit-learn numpy

2. Run the server:
python server.py

3. Open your browser and go to:
http://localhost:5000
```
